# UNSENT

> A global anonymous emotion-sharing platform.

## Overview
UNSENT is a real-time platform where users can share their feelings anonymously. Messages are visualized as stars on a 3D map, creating a collective universe of human emotion. The application features real-time connection through "Knot Sessions" and leverages AI for emotion detection.

## Tech Stack
- **Frontend**: Next.js (React)
- **Backend**: Flask (Python)
- **Database**: Supabase (PostgreSQL)
- **Real-time**: Socket.io
- **AI**: TextBlob / Transformers

## Branch Workflow
**STRICT RULES:**
- `main` is PROTECTED. No direct commits allowed.
- `develop` is the primary integration branch.
- Feature branches must follow the pattern `feature/<feature-name>`.

### Workflow Steps:
1.  **Branch off** from `develop`: `git checkout -b feature/my-feature develop`
2.  **Commit** work: `git commit -m "feat: description"`
3.  **Push** to origin: `git push origin feature/my-feature`
4.  **Create PR** into `develop`.
